Downloading Postgres-XC

List of Release Materials

The current release includes the following materials.
Please note that documentation is not included in the source material.
Please download documentation from the project download page.

Please also note tarball files do not include Postgres-XC documents.

Version 0.9.5

  • pgxc_v0.9.5.tar.gz: 
    Latest version of Postgres-XC available.
    Please note that Postgres-XC documentation is not included in this file. ⇒ (download)
  • PGXC_v0_9_5-PG_REL9_0_4.patch.gz: 
    The same material as above, but this file includes only the patch to apply to the PostgreSQL 9.0.4 release source code.
    It is useful if you would like to see just a difference between PostgreSQL and Postgres-XC.
    No Postgres-XC documentation is included in this file either. ⇒ (download)
  • COPYING: 
    License description. Postgres-XC is distributed under the PostgreSQL license. ⇒ (download)
  • FILES: 
    Description of files included in Postgres-XC 0.9.5 release. ⇒ (download)
  • PG-XC_ReferenceManual_v0_9_5.pdf: 
    Reference of Postgres-XC extension. ⇒ (download)
  • PG-XC_pgbench_Tutorial_v0_9_5.pdf: 
    Step by step description how to build and configure pgbench to run with Postgres-XC. ⇒ (download)
  • PG-XC_DBT1_Tutorial_v0_9_5.pdf: 
    Step by step description how to build and configure DBT-1 to run with Postgres-XC. ⇒ (download)
  • PG-XC_InstallManual_v0_9_5.pdf: 
    Step by step description how to build, install and configure Postgres-XC. ⇒ (download)
  • PG-XC_SQL_Limitations_v0_9_5.pdf: 
    SQL restrictions available for Postgres-XC 0.9.5. ⇒ (download)
