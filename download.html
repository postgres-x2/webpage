Downloading Postgres-XC

List of Release Materials

The current release includes the following materials. Please note that documentation is not included in the source material. Please download documentation from the project download page.

Please also note tarball files do not include Postgres-XC documents.

Version 0.9.1

  • pgxc_v0.9.1.tar.gz:  Latest version of Postgres-XC available. Please note that Postgres-XC documentation is not included in this file. ⇒ (download)
  • PGXC_v0_9_1-PG_REL8_4_3.patch.gz:  The same material as above, but this file includes only the patch to apply to the PostgreSQL 8.4.3 release source code. It is useful if you would like to see just a difference between PostgreSQL and Postgres-XC. No Postgres-XC documentation is included in this file either. ⇒ (download)
  • COPYING:  License description. Postgres-XC is distributed under LGPL version 2.1 ⇒ (download)
  • FILES:  Description of files included in Postgres-XC 0.9.1 release. ⇒ (download)
  • README:  Overview of the release. ⇒ (download)
  • PG-XC_ReferenceManual_v0_9_1.pdf:  Reference of Postgres-XC extension. ⇒ (download)
  • PG-XC_pgbench_Tutorial_v0_9_1.pdf:  Step by step description how to build and configure pgbench to run with Postgres-XC. ⇒ (download)
  • PG-XC_DBT1_Tutorial_v0_9_1.pdf:  Step by step description how to build and configure DBT-1 to run with Postgres-XC. ⇒ (download)
  • PG-XC_InstallManual_v0_9_1.pdf:  Step by step description how to build, install and configure Postgres-XC. ⇒ (download)
  • PG-XC_Architecture_v0_9.pdf:  Description of the outline of Postgres-XC internals. ⇒ (download)
