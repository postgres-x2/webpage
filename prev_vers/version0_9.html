Downloading Documents of previous versions of Postgres-XC

List of Previous Release Materials

Version 0.9.0

  • pgxc_v0.9.tar.gz: 
    Previous version of Postgres-XC released in April 2010.
    This is a collection of source materials used to build the binaries.
    Please note that Postgres-XC documentation is not included in this file. ⇒ (download)
  • PGXC-PG_REL8_4_3.patch.gz: 
    The same material as above, but this file includes only the patch to apply to the PostgreSQL 8.4.3 release source code.
    It is useful if you would like to see just a difference between PostgreSQL and Postgres-XC.
    No Postgres-XC documentation is included in this file either. ⇒ (download)
  • COPYING: 
    License description. Postgres-XC is distributed under LGPL version 2.1 ⇒ (download)
  • README: 
    Overview of the release. ⇒ (download)
  • ReferenceManual.pdf: 
    Reference of Postgres-XC extension. ⇒ (download)
  • PG-XC_TutorialManual.pdf: 
    Step by step description how to build and configure DBT-1 to run with Postgres-XC. ⇒ (download)
  • PG-XC_InstallManual_Revision1.pdf: 
    Step by step description how to build, install and configure Postgres-XC. ⇒ (download)
  • PG-XC_Architecture.pdf: 
    Description of the outline of Postgres-XC internals. ⇒ (download)
